InfoGB Win32 v0.5J

はじめに

　InfoGB Win32は、Win32で動作するゲームボーイ(TM)のエミュレータです。

更新状況

v0.5J (03/11/24)

  • 初期リリース
    • Win32: ウェイトコードの実装
    • Win32: サウンドコード（DirectSoundを利用）を実装
    • Win32: ソースコードをCからC++に変更
    • Win32: 関数名等を修正
    • Win32: バージョン表示を修正
    • Win32: 著作権表示を修正
    • Win32: ウィンドウ上のアイコン表示を修正
    • Win32: アイコン表示を修正
    • Win32: ファイル選択機能を追加
    • Win32: ドラッグ＆ドロップ機能を追加
    • Win32: Win32に依存した関数群を実装
    • Win32: GBE v0.4.8をVC++6.0に移植

動作環境

  • OS: Windows XP
  • CPU: Intel Celeron 600 MHz
  • メモリ: 256MB
  • SW: DirectX 7.0a（以上）

操作方法

コントローラ1

  • [Enter] : スタート
  • [Tab] : セレクト
  • [Space] : Aボタン
  • [Ctrl] : Bボタン
  • [↑] : 上ボタン
  • [↓] : 下ボタン
  • [→] : 右ボタン
  • [←] : 左ボタン
